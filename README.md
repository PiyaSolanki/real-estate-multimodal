# Real Estate Multimodal Price Prediction

## Overview
This project builds a machine learning pipeline to predict house prices using tabular real estate data.
A Random Forest baseline model is implemented before extending the approach to include satellite imagery.

## Files
- preprocessing.ipynb: Exploratory data analysis and preprocessing, including log-price transformation.
- model_training.ipynb: Training and evaluation of a Random Forest regression model.

## Model Details
- Algorithm: RandomForestRegressor
- Target: Log-transformed house price
- Metrics: RMSE, RÂ²
